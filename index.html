Facebook Marketplace Filter

What does it do?

The Facebook Marketplace is great, but the way it sorts listings is awful. Unfortunately, I can't change the way it sorts, but I can change the way it looks. This is an extension for Google Chrome that will fade out older listings.

How do I use it?

Once installed, you will have a small Facebook-like icon in your toolbar. When visiting the FB Marketplace, simply click the icon.

Clicking the icon will insert two options in the blue bar at the top. This lets you show/hide listings that were posted days or weeks ago.

Sounds good. How do I install it?

Because I'm too lazy to publish it through the Chrome Web Store, you have to download the file to your computer.

Once downloaded, visit chrome://extensions and then drag & drop the downloaded file onto the page.
